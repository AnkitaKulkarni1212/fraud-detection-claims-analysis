# Claims Management Fraud Detection Analysis

## Overview
This project analyzes a claims management fraud case involving manipulated insurance reimbursement data at Reliability Tire, Inc.

The objective was to identify fraud risks, analyze claim workflows, and recommend internal controls to prevent financial loss.

## Problem Statement
An internal audit revealed suspicious claim reimbursements caused by modified claims data, leading to significant financial impact.

## Methodology
- Fraud risk assessment
- Claims workflow analysis
- Stakeholder interview framework
- Data validation checkpoints
- Control and approval process evaluation

## Key Analysis Areas
- Claims submission validation
- Retailer data verification
- Fraud risk identification
- Approval workflow analysis
- Internal audit recommendations

## Tools & Skills Demonstrated
- Fraud Analytics
- Risk Assessment
- Business Process Analysis
- Data Validation
- Forensic Accounting Concepts

## Outcome
Proposed structured fraud prevention controls and verification mechanisms to reduce manipulation risks in claims processing systems.
